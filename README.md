# LIS
Functional Analysis of LIS

# Business Problem
The laboratory team is facing delays in sample processing and report generation because multiple steps in the workflow were partially manual and disconnected across systems.

Lab technicians are manually updating sample status in different applications, which has caused:

duplicate entries,
tracking errors,
delayed test result availability,
and a lack of real-time visibility for stakeholders.

Additionally, there is inconsistent integration between the Laboratory Information System (LIS) and downstream reporting systems, increasing turnaround time and creating operational inefficiencies.
